queryConnectionNumber()

queryConnectionNumber() : integer

Queries the number of current connections to the database for which the connection has been established.

If the fetched row is false, means that the query has returned no row, so, that means that the database has not any connection. Which is certainly impossible since this tool is connected to the database to make the query.

As the connection query will also count the connection made by this tool to make that query, and that this connection can be considered as "residual", it is subtracted from the connection count. For example, if the tool is configured for a threshold of 1 connection (which would be weird), the tool would always return 1 if its connection is not subtracted (which would be even more weird, since the database is not having a real usage).

Throws

\DBConnectionWatcher\DB\PreparedStatementCreationException

If an error occurs creating the prepared statement.

Returns

integer —

The number of connections.
